# Projeto Sprint 5

# Dashboard de Vendas de Carros

Este é um aplicativo web interativo criado com Streamlit para explorar dados de anúncios de vendas de carros.

## Funcionalidades
- Visualização de histogramas da quilometragem dos veículos
- Gráfico de dispersão mostrando a relação entre quilometragem e preço
- Botões e caixas de seleção para interatividade

## Como executar
```bash
streamlit run app.py
```
